# DKT Marketing Landing Pages

Simple static site for the DKT Marketing landing pages.

Files:
- `index.html` — Home / Landing page
- `about.html` — About page
- `our-work.html` — Portfolio / case studies
- `contact.html` — Contact form page
- `css/styles.css` — Styling and responsive rules

To preview locally, open `index.html` in your browser or run a simple static server.

Example (PowerShell):
```powershell
# serve current folder on port 8000 (Python must be installed)
python -m http.server 8000 --bind 127.0.0.1
```
